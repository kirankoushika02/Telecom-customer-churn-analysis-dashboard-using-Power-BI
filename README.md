# Telecom-customer-churn-analysis-dashboard-using-Power-BI

## Overview
This project presents an interactive Power BI dashboard designed to analyse customer churn patterns and identify factors influencing customer retention. The dashboard helps uncover trends in churn behaviour, customer segments, and key performance indicators that support data-driven business decisions.

## Objectives
- Analyse customer churn trends and behaviour
- Identify high-risk customer segments
- Track important KPIs related to churn
- Build an interactive dashboard for business insights
- Support retention strategy planning through data visualisation

## Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV dataset

## Dashboard Features
- Overall churn rate summary
- Customer segmentation by demographics or service type
- Churn comparison across categories
- KPI cards for key metrics
- Interactive filters and slicers
- Trend and distribution visualisations

## Key Insights
- Identifies the categories with the highest churn
- Highlights customer groups that are more likely to leave
- Supports better understanding of retention challenges
- Helps stakeholders monitor churn patterns effectively
